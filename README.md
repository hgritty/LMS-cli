# 图书管理系统

## 项目简介
一个基于 C++ 的命令行图书管理系统，支持增删改查和借阅管理。

## 已完成功能
- [x] 添加图书
- [x] 查看所有图书
- [x] 借阅图书
- [x] 归还图书
- [x] 数据持久化（保存/加载）

## 待开发功能
- [ ] 删除图书
- [ ] 按书名搜索
- [ ] 按作者搜索
- [ ] 统计功能（总书数、已借数量）
- [ ] 修改图书信息
- [ ] 同一本书多副本支持

## 开发环境
- C++11
- Ubuntu 24.04 / WSL2

## 编译运行
```bash
g++ -std=c++11 library.cpp -o lib
./lib
```

## 数据格式
数据保存在 books.txt 中，格式为：

```text
编号|书名|作者|状态
1|C++ Primer|Lippman|0
```
其中 0 表示在库，1 表示已借出。
